ID en HTML y CSS

ID

Un ID en HTML y CSS, se define como un atributo de un elemento único (id)
Cuando vayamos a usar un id, NO debería de haber otro elemento del mismo
nombre del idenfitificador (id) dentro del documento HTML.
En el html tendremos que poner id="nombre", y en CSS #nombre.

Class

Un Class en HTML y CSS, se define como un atributo, que a diferencia de un id
puede ser utilizado en más de un elemento de un documento HTML.
En el html tendremos que poner class="nombre", y en CSS .nombre
