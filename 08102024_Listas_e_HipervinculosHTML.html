Listas en HTML5

Una lista en HTML, se utiliza para listar contenido en nuestra página web.
Existen tres tipos de Listas
  • Listas Ordenadas: vienen representadas por la etiqueta ol (Ordered List)
  • Listas Desordenadas: vienen representadas por la etiqueta ul (Unordered List)
  • Listas de Definición: vienen representadas por la etiqueta dl (Definition List)

Dentro de un elemento lista, se puede insertar cualquier tipo de información:
texto, imágenes, tablas, listas, etc.

Listas Ordenadas

Las listas ordenadas se caracterizan por ir en orden el contenido que se le introduce.
Su principales características son las siguientes.
  • Está delimitada por la etiqueta de apertura y cierre ol
  • Internamente, tendrá una etiqueta li de apertura y cierre,
    en donde irá el contenido a listar.
  • Podremos cambiar el tipo de listado a través de la definición de
    sus atributos.

Ejemplo de Lista Ordenada (ol)
  1. Linux Mint
  2. Ubuntu
  3. Debian
  4. Windows 11/10
  5. Kali Linux

Listas Desordenadas

Las listas desordenadas se caracterizan por ir listadas sin un orden específico del
contenido que se le introduce. Es decir, no van enumeradas como ocurre con las
listas ordered list (ol)
Su principales características son las siguientes.
  • Está delimitada por la etiqueta de apertura y cierre ul
  • Internamente, tendrá una etiqueta li de apertura y cierre,
    en donde irá el contenido a listar.
  • Podremos cambiar el tipo de listado a través de la definición de
    sus atributos.

Ejemplo de Lista Desordenada (ul)
  • FHW
  • ISO
  • PAR
  • GBBDD
  • Fundamentos Cloud

Cambio de Forma en Listas Ordenadas y Desordenadas

La diferencia entre las listas ordenadas y desordenadas, es que los navegadores
numeran (con letras o números) los elementos de las listas ordenadas, mientras
que en las listas desordenadas se dibujan símbolos gráficos.
Para poder cambiar esto, utilizaremos el atributo type seguido del elemento
a enumerar u ordenar, y el atributo start para indicar el valor numérico
con el que queremos comenzar. También podemos usar el atributo
type=forma para establecer la forma gráfica de nuestra lista.
  • A(mayúsculas) y a (minúsculas)
  • I(cifras romanas mayúsculas) y i(cifras romanas minúsculas)
  • 1(para números predeterminados)
  • disc (viñetas redondas rellenas)
  • circle (viñetas redondas vacías)
  • square (viñetas cuadradas)

Listas de Definición

Las lista de definición, tienen una estructra distinta a las listas ordenadas y desordenadas.
Su estructura es la siguiente:
  • La primera etiqueta que tenemos es la de apertura y cierre
    llamada dl. Esta etiqueta significa Description List
  • A continuación, tenemos la etiqueta de apertura y cierre dt
    Significa Description Term
  • Por último, tenemos la etiqueta dd, que significa
    Definition Description.

Ejemplo de etiqueta de Definición
dl
Crea la caja que contiene los items de la lista
dt
Item que contiene un término a definir
dd
Item que contiene una definición

Hipervínculos o Hiperenlaces en HTML5

Son elementos que se caracterizan en la web porque nos permitirá relacionar las
páginas web con otras páginas o con contenido de un mismo documento.
Un hiperenlace o hipervínculo, tiene dos partes:
  • Es el enlace. Es considerado el elemento (texto o imagen) sobre el que
    haremos click con el rató.
  • Es destino. Es considerado como el elemento (página web, imagen,
    archivo, etc.), que se mostrará cuando hagamos click en el enlace.

La estructura de un hiperenlace o hipertexto, se caracteriza por utilizar una
etiqueta que se llama Archor y que viene definida por la etiqueta
a.
La estructura que presenta es la siguiente:
  • Comenzamos con la etiqueta de apertura a
  • Insertamos el atributo href, que es donde insertaremos la ruta
    donde queremos que cuando pinchemos en el enlace, nos lleve.
  • Cerramos la etiqueta de apertura, insertamos el texto o imagen
    que queremos vincular al hipervínculo, y por último insertamos la
    la etiqueta de cierre de a.

En HTML, existe un atributo, que nos permite poder abrir el hipervínculo o hiperenlace en otra pestaña de
nuestro navegador.
El atributo en cuestión es target=_blank

Insertar correo en HTML5

Para insertar un correo en HTML, utilizaremos la misma etiqueta a, pero seguido del atributo mailto
y del correo al que queremos linkear nuestro mensaje.
Estructura: 
< a href = "mailto:raul.rodriguez@univeridadeuropea.es" >
                        Pincha aquí para contactar conmigo
                        < / a >
