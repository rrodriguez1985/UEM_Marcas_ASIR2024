Tablas en HTML

Las tablas en HTML, se podrían definir como un conjunto de celdas en donde vamos a
poder insertar cualquier tipo de información: texto, imágenes, multimedia,
formularios, otras tablas...
La estructura de a la hora de crear tablas será la siguiente:
  • table: la etiqueta table, tiene apertura y cierre, y sirve para
    definir que vamos a utilizar/crear una tabla en HTML.
  • tr: es la etiqueta de apertura y cierre, que vamos a utilizar para
    definir las filas de nuestra tabla.
  • td: es la etiqueta de apertura y cierre, que vamos a utilizar para
    definir las celdas que vamos a tener en nuestra tabla (columnas)
  • Recordar: en una tabla, podemos insertar cualquier tipo de información /
    elemento: texto, imágenes, vídeos, audios, formularios, tablas, etc.

Propiedades que podemos usar en las tablas

  • table: es la etiqueta que nos va a indicar que vamos a utilizar
    una tabla en HTML.
  • caption: es para poner un título en una tabla
    Podemos utilizar la alineación align=top o align=bottom.
  • thead o th: Define la cabecera de una tabla
  • tbody: Define el contenido de la celda de una tabla
  • tfoot: Define el pie de la tabla
  • tr: Define la fila de una tabla
  • td: Define la celda de una fila de la tabla, es decir,
    las columnas de una tabla
  • border: indica el grosor del borde de la tabla en px

Combinación de celdas en tablas

A la hora de diseñar tablas, podemos unificar celdas de manera tanto horizontal como
de manera vertical.
Para ello, utilizaremos las siguientes propiedades:
  • rowspan: se utiliza para indicar el número de columnas que ocupan
    cada celda. Es decir, combinación de celdas en vertical.
  • colspan: se utiliza para indicar el número de filas que ocupan
    cada celda. Es decir, combinación de celdas en horizontal.

A parte del rowspan y colspan, en las celdas, podemos aplicar los atributos
cellpadding y cellspacing.
  • cellpadding: especifica el espacio entre el borde de una celda
    y su contenido.
  • cellspacing: especifica el espacio entre celdas
