Mapeado de Imágenes en HTML

El mapeado de imágenes en HTML, sirve para especificar mediante una región geométrica
de un mapa, un vínculo asociado a una cierta región.
Esto va a asociado a una serie de etiquetas y atributos que son los siguientes:
  • usemap: este atributo sirve para indicar en la foto,
    que vamos a utilizar un mapeado y le damos un nombre.
  • map y name: Para establecer que vamos a utilizar un mapeado
  • area: se utiliza para establecer la zona de mapeado de la imagen
  • coords: establecemos las coordenadas de la región o mapeado que vamos a utilizar
  • shape: es definir el tipo de región o mapeado que vamos a utilizar.
    • default: cuya región de mapeado ocupa toda la región o espacio de imagen (región completa).
    • rect: rectangular. Se establece con un punto X1,Y1 que representa la esquina superior izquierda
      y un punto X2, Y2 que representa la esquina inferior derecha.
    • circle: circunferencia. Se establece con un punto central X1, Y1, y un valor de radio.
    • poly: poligonal. Se establece como mínimo 3 puntos o más para establecer un polígono de región del área.
      Estos puntos, se establecen siguiendo el sentido de las agujas del reloj, y no siendo necesario
      cerrar el último punto.
